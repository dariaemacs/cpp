# C++ tutorial

- ## move: 
  ### как писать перемещающие конструкторы и перемещающие операторы присваивания в классах с указателями?

  Пример в папке: ```cpp/move/```
  
  ```sh 
  $ git clone git@github.com:dariaemacs/cpp.git
  $ cd cpp
  $ cmake -S . -B build
  $ cmake --build build
  $ build/move/./move
  ```
